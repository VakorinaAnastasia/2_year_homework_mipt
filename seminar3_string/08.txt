После ввода числа в буфере ввода остаётся символ переноса строки \n. Формат %[^\n] в функции scanf читает все символы до \n, но не поглощает сам символ \n. Поэтому следующий вызов scanf("%[^\n]", str) сразу натыкается на оставшийся \n и завершается, не записывая ничего в строку str.

#include <stdio.h>
int main()
{
    int a;
    scanf("%i", &a);
    printf("a = %i\n", a);

    char str[100];
    scanf(" %[^\n]", str);  // добавить нужно пробел перед %
    printf("str = %s\n", str);
}
